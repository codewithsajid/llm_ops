The policy and value function are two key concepts in reinforcement learning. The policy function maps from state to action, while the value function estimates the future reward for each state. Although they are related, they are distinct functions. For example, a deterministic policy specifies a single action for each state, while a stochastic policy assigns a probability to each possible action.
